while (i >= 0 || j >= 0 || carry != 0)含义：
字符串 a 和 b 只要有一个没遍历完，那么就继续遍历；
如果字符串 a 和 b 都遍历完了，但是最后留下的进位 carry != 0，那么需要把进位也保留到结果中。
取 digit 的时候，如果字符串 a 和 b 中有一个已经遍历完了（即 i <= 0或者 j <= 0），则认为 a 和 b 的对应位置是 0 。
